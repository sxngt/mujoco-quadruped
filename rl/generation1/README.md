# Generation 1: 기초 환경 구축

## 개요
초기 단계에서는 기본적인 강화학습 환경 구축과 물리 시뮬레이션 문제 해결에 집중했습니다.

## 주요 특징
- **기본 PPO 구현**: 표준 Proximal Policy Optimization 알고리즘
- **물리 환경 구축**: 지면, 중력, 마찰력 등 기본 물리 요소 추가
- **텐서 차원 문제 해결**: PyTorch 텐서 관리 안정화

## 주요 문제점
- 로봇이 공중에 떠 있는 무중력 상태
- 텐서 차원 불일치로 인한 학습 중단
- 지면과의 상호작용 부재

## 해결 방안
- 지면 geometry 추가
- NumPy 배열을 통한 안전한 텐서 변환
- 물리 파라미터 정확한 설정
